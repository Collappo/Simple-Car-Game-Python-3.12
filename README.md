# Simple CAR game in Python
------------------------------------------
# 1) **Driving a car**:
  - `A` - go left.
  - `D` - go right.
  - `W` - to accelerate.
  - `S` - to deaccelerate
  ------------------------------------------
# 2) **Installation**:
  1) **Download and install packages**:
     
    pip install -r requirements.txx
    
  or

    pip install pandas
    pip install keyboard`
